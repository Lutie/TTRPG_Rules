# Chapitre 11 : L'Adversité

## Généralités

L'adversité désigne l'ensemble des opposants, créatures et entités auxquels les personnages peuvent être confrontés. Le terme est volontairement neutre : il couvre aussi bien les humanoïdes que les animaux, les monstres, les machines ou toute autre forme d'antagoniste.

Ce chapitre regroupe les règles permettant de créer, qualifier et faire jouer ces adversaires. Les règles de confrontation restent inchangées — c'est la nature de l'adversaire qui détermine quelles règles s'y appliquent et comment les adapter.
