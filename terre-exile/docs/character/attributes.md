# Attributes

Attributes are meant to change a bit throughout the character's journey by progressing in their pathes.  
Average (and base) value is 25 for humanoïds.  
All attributes have a passive improvement equal to the character's level.

### Strength (STR)

This a primary attribute.  
One of the attribute's of the body.  
Attached to fire element.

|                                                                                      |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <div style="width:200px;"><img src="../../images/attributes/strength.webp" width="200px"></div> | Strength represents a character’s raw physical power, though it doesn’t directly indicate muscle mass. A character can be strong without an imposing stature; Stature better reflects body mass. Strength is crucial for wielding impact weapons, such as hammers and axes, where a powerful swing makes all the difference. It also supports demanding physical tasks like lifting heavy objects, breaking down barriers, and climbing steep surfaces. Beyond brute force, Strength enhances physical skills and interactions like blacksmithing and intimidation, where a solid physique becomes a key asset. |
