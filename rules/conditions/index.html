    Conditions

    Conditions positives

    Note : une condition positive peut être acceptée volontairement sans effectuer de sauvegarde.

    Ire

    • Type : Positif
    • Domaines : 🌪️ Air / ⚔️ Acier
    • Sauvegarde : Sans
    • Description : Une colère froide s’empare du personnage. Sa respiration se fait courte, ses gestes plus secs et plus précis, chaque attaque étant portée avec une agressivité parfaitement maîtrisée.
    • Effet : (double) Avantage aux tests d'attaques

    Prudence

    • Type : Positif
    • Domaines : 🪨 Terre / ⚔️ Acier
    • Sauvegarde : Sans
    • Description : Le personnage adopte une posture ancrée et défensive. Il anticipe les coups, réduit les ouvertures et transforme chaque mouvement en une mesure de protection.
    • Effet : (double) Avantage aux tests de défense

    Fluide

    • Type : Positif
    • Domaines : 💧 Eau / ⚔️ Acier
    • Sauvegarde : Sans
    • Description : Le personnage s’adapte au rythme du combat. Ses décisions coulent naturellement d’une action à l’autre, exploitant chaque opportunité avec souplesse.
    • Effet : (double) Avantage aux tests de tactique

    Réactif

    • Type : Positif
    • Domaines : 🐗 Faune / ⚔️ Acier
    • Sauvegarde : Sans
    • Description : Les instincts du personnage prennent le dessus. Son corps réagit avant même que l’esprit n’analyse, déclenchant des réponses immédiates et efficaces.
    • Effet : (double) Avantage aux tests d'initiative

    Résilient

    • Type : Positif
    • Domaines : ✨ Sacre / ❤️ Vie
    • Sauvegarde : Sans
    • Description : Une force bienveillante protège le personnage. Les coups semblent perdre de leur gravité, comme si son corps refusait de céder sous la douleur.
    • Effet : Les jets subit sont réduits de 1d6 (2d6)

    Héroïsme

    • Type : Positif
    • Domaines : ✨ Sacre / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Animé par une foi ardente, le personnage accomplit des exploits hors du commun. Ses réussites prennent une ampleur héroïque, dépassant ses limites habituelles.
    • Effet : Les jets infligés sont augmentés de 1d6 (2d6)

    Aisance

    • Type : Positif
    • Domaines : ✨ Sacre / 🪷 Nature
    • Sauvegarde : Sans
    • Description : Le personnage agit avec une aisance surnaturelle. Chaque geste, chaque parole semble juste, comme s’il était guidé par une harmonie intérieure.
    • Effet : Tous les tests reçoivent un ajustement positif de 2 (3)

    Vigilance

    • Type : Positif
    • Domaines : 💧 Eau / 👁️ Vision
    • Sauvegarde : Sans
    • Description : L’attention du personnage est constamment aiguisée. Il repère la moindre menace et ajuste instinctivement ses défenses.
    • Effet : Défenses passives +2 (3)

    Vigueur

    • Type : Positif
    • Domaines : ☀️ Lumière / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : Une énergie lumineuse irrigue le corps du personnage, renforçant son endurance et sa capacité à encaisser les épreuves physiques.
    • Effet : (double) Avantage aux tests de Robustesse

    Confort

    • Type : Positif
    • Domaines : ☀️ Lumière / 🧠 Esprit
    • Sauvegarde : Sans
    • Description : Le personnage évolue dans un état de stabilité rassurante. Cette sérénité renforce sa détermination et sa capacité à tenir face à la pression.
    • Effet : (double) Avantage aux tests de Détermination

    Vif

    • Type : Positif
    • Domaines : ☀️ Lumière / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : Le corps du personnage répond avec une vivacité éclatante. Ses réflexes sont plus rapides, réduisant toute hésitation.
    • Effet : (double) Avantage aux tests de Réflexes

    Tenace

    • Type : Positif
    • Domaines : ☀️ Lumière / 🧠 Esprit
    • Sauvegarde : Sans
    • Description : Une volonté inébranlable soutient le personnage. Il reste calme et lucide, même face aux situations les plus éprouvantes.
    • Effet : (double) Avantage aux tests de Sang Froid

    Rébellion

    • Type : Positif
    • Domaines : 🛡️ Guerre / 🌀 Chaos
    • Sauvegarde : Sans
    • Description : Le personnage étend son influence sur le champ de bataille, défiant l’ordre établi et imposant une pression constante autour de lui.
    • Effet : Zone +2 (3) et Allonge/portée +1 (2)

    Solide

    • Type : Positif
    • Domaines : ✨ Sacre / 🐗 Faune
    • Sauvegarde : Sans
    • Description : Le corps du personnage devient un rempart. Les tentatives de l’affaiblir échouent plus souvent que prévu.
    • Effet : Tests de sauvegades +2 (3)

    Tonique

    • Type : Positif
    • Domaines : ❤️ Vie / ☀️ Lumière
    • Sauvegarde : Sans
    • Description : Une énergie vitale circule en permanence, régénérant le personnage et lui permettant de soutenir l’effort tour après tour.
    • Effet : Génère 4 (7) PE par tour

    Sanguinaire

    • Type : Positif
    • Domaines : 🩸 Impie / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Chaque attaque réussie alimente une soif de combat. La violence devient une source directe de puissance et d’endurance.
    • Effet : Génère 4 (7) PE à chaques attaques réussies infligés

    Invulnérable

    • Type : Positif
    • Domaines : ✨ Sacre / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Chaque défense victorieuse renforce la protection du personnage, comme si tenir bon nourrissait sa puissance intérieure.
    • Effet : Génère 4 (7) PE à chaques défenses réussies

    Infaillible

    • Type : Positif
    • Domaines : 🛡️ Guerre / ✨ Sacre
    • Sauvegarde : Sans
    • Description : Le personnage devient un bastion mobile. Les attaques glissent sur sa protection renforcée.
    • Effet : Protection +3 (5)

    Liberté

    • Type : Positif
    • Domaines : 🌪️ Air / 🌀 Chaos
    • Sauvegarde : Sans
    • Description : Le personnage se déplace sans contrainte, brisant les entraves physiques ou mentales qui pourraient freiner son allure.
    • Effet : Allure +3 (5)

    Hyperactif

    • Type : Positif
    • Domaines : ⚡ Foudre / 💢 Vide
    • Sauvegarde : Sans
    • Description : Une agitation électrique parcourt le corps du personnage, accélérant ses réactions et son initiative.
    • Effet : Initiative +3 (5)

    Rapide

    • Type : Positif
    • Domaines : 🌪️ Air / 💢 Vide
    • Sauvegarde : Sans
    • Description : Les mouvements du personnage gagnent en vitesse pure, lui permettant d’agir plus rapidement que la normale.
    • Effet : Rapidité +3 (5)

    Vivace

    • Type : Positif
    • Domaines : 🌪️ Air / 💢 Vide
    • Sauvegarde : Sans
    • Description : Une vitalité bondissante anime le personnage, augmentant durablement sa vitesse et sa capacité à enchaîner les actions.
    • Effet : Vitesse +3 (5)

    Assurance

    • Type : Positif
    • Domaines : ⚖️ Loi / 🧩 Mental
    • Sauvegarde : Sans
    • Description : Le personnage agit avec une confiance méthodique. Les erreurs sont réduites et les actions critiques mieux maîtrisées.
    • Effet : Seuil de maladresse +3 (5) et criticité -3 (5)

    Résilience

    • Type : Positif
    • Domaines : 🐗 Faune / ✨ Sacre
    • Sauvegarde : Sans
    • Description : Le personnage supporte mieux les blessures. La douleur est atténuée et les pénalités deviennent moins pénalisantes.
    • Effet : Pour déterminer les pénalités issues des blessures la gravité est réduite de ¼ ou 5, le plus haut des deux

    Prompt

    • Type : Positif
    • Domaines : 💢 Vide / 🌪️ Air
    • Sauvegarde : Sans
    • Description : Guidé par l’instinct et la vivacité, le personnage accélère son allure et se repositionne avec une efficacité animale.
    • Effet : Allure augmentée de 3 (5), tests via les jambes +1 (2)

    Célérité

    • Type : Positif
    • Domaines : 🌪️ Air / ⚡ Foudre
    • Sauvegarde : Sans
    • Description : Les actions rapides s’enchaînent avec fluidité, réduisant leur coût et améliorant l’initiative globale.
    • Effet : Action rapides cout -2 (3) initiative

    Renfort

    • Type : Positif
    • Domaines : ⚡ Foudre / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Une impulsion d’énergie galvanise un allié, lui permettant d’agir au-delà de ses capacités habituelles.
    • Effet : Le personnage peux réaliser une action simple de plus par tour, mais la limite des actions similaires l'applique malgré tout (s'applique à tous les tours à la place)

    Volubile

    • Type : Positif
    • Domaines : ⚕️ Corps / 🧠 Esprit
    • Sauvegarde : Sans
    • Description : Les paroles du personnage portent naturellement. Il se fait comprendre avec clarté et efficacité.
    • Effet : Le personnage a un (double) avantage à se faire comprendre / à convaincre

    Acuité

    • Type : Positif
    • Domaines : 👁️ Vision / ☀️ Lumière
    • Sauvegarde : Sans
    • Description : Les sens visuels sont aiguisés à l’extrême, permettant de remarquer des détails invisibles aux autres.
    • Effet : Le personnage a un avantage aux actions relevant de la vue (où la vue a un rôle majoritaire, tel que l'observation mais pas le combat)

    Joie

    • Type : Positif
    • Domaines : 🪷 Nature / ⚜️ Charme
    • Sauvegarde : Sans
    • Description : Une exaltation sincère accompagne le combat, favorisant les exploits tout en réduisant les maladresses.
    • Effet : Plage d'exploit et maladresse des actions de combat +1 (2) et -1 (2)

    Sanctification

    • Type : Positif
    • Domaines : ✨ Sacre / ⚖️ Loi
    • Sauvegarde : Sans
    • Description : Une bénédiction totale enveloppe le personnage, élevant l’ensemble de ses capacités physiques et mentales.
    • Effet : Tous les attributs sont augmentés de 1 (2)

    Accalmie

    • Type : Positif
    • Domaines : ❄️ Glace / ❤️ Vie
    • Sauvegarde : Sans
    • Description : Le personnage devient impérméable aux affres de ses blessures et autres formes de ruptures.
    • Effet : Les dégats de ruputure que subit le personnage sont divisé par 2 (annulé)

    Impétueux

    • Type : Positif
    • Domaines : 💧 Eau / ❤️ Vie
    • Sauvegarde : Sans
    • Description : Le personnage agit avec une ardeur inépuisable, ignorant la fatigue et continuant d’avancer comme si l’épuisement n’avait aucune prise sur lui.
    • Effet : Le personnage doit avoir des PE négatif équivalant à la moitié de son maximum avant d'être considéré comme choqué/épuisé (ne l'est jamais, quelque soit les PE), ses défenses augmentent de 1 (2)

    Colère

    • Type : Positif
    • Domaines : ⚖️ Loi / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Le personnage agit sous une colère rigide et autoritaire, concentrant toute son efficacité sur l’attaque au détriment de sa prudence et de sa capacité d’analyse.
    • Effet : Le test d'attaque est augmentés de 3 (4) mais les défenses/tactiques sont réduites de 1 (2)

    Réserve

    • Type : Positif
    • Domaines : ⚖️ Loi / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Le personnage se replie dans une discipline stricte, renforçant sa défense par le contrôle et la retenue, mais sacrifiant son potentiel offensif et tactique.
    • Effet : Le test de défense est augmentés de 3 (4) mais les attaques/tactiques sont réduites de 1 (2)

    Attention

    • Type : Positif
    • Domaines : ⚖️ Loi / 🛡️ Guerre
    • Sauvegarde : Sans
    • Description : Le personnage focalise toute son attention sur l’analyse et la planification, améliorant ses choix tactiques tout en négligeant attaque et défense.
    • Effet : Le test de tactique est augmentés de 3 (4) mais les attaques/défenses sont réduites de 1 (2)

    Acuité

    • Type : Positif
    • Domaines : 👁️ Vision / ☀️ Lumière
    • Sauvegarde : Sans
    • Description : Les perceptions du personnage atteignent un niveau de précision exceptionnel. Il anticipe les dangers avant qu’ils ne se manifestent pleinement, renforçant ses défenses passives, réagissant avec une grande efficacité en défense active et libérant ses décharges avec une intensité accrue.
    • Effet : Défenses passives augmentées 2 (3) et défenses actives également

    6e Sens

    • Type : Positif
    • Domaines : 👁️ Vision / 🧩 Mental
    • Sauvegarde : Sans
    • Description : Les sens du personnage dépassent la perception ordinaire. Sons, mouvements et présences sont perçus avec une acuité instinctive, lui conférant un avantage constant dès qu’une situation repose sur l’observation ou la perception sensorielle.
    • Effet : Le personnage reçoit un avantage à tous les tests nécessitans ses sens

    Gracieux

    • Type : Positif
    • Domaines : ✨ Sacre / ❤️ Vie
    • Sauvegarde : Sans
    • Description : Le personnage agit sous une grâce sacrée subtile mais constante. À chaque tour, son premier effort est naturellement favorisé, comme si une bénédiction guidait précisément ce moment clé de son action.
    • Effet : Le personnage reçoit un bonus de 3 à un seul de ses tests, le premier

    Coagulé

    • Type : Positif
    • Domaines : 🔥 Feu / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : La chaleur interne du personnage provoque une coagulation rapide des blessures. Les ruptures sont partiellement refermées ou stabilisées, réduisant sensiblement leur gravité et leurs effets.
    • Effet : Les ruptures infliges 1/3 de moins (1/2)

    Stable

    • Type : Positif
    • Domaines : 🪨 Terre / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : Le personnage est solidement ancré, physiquement et mentalement. Il résiste aux tentatives de le déséquilibrer ou de le déplacer, maintenant sa position avec une stabilité remarquable.
    • Effet : Le personnage est avantage à tous ses tests visants à garder sa position (équilibre, etc).

    Fortitude

    • Type : Positif
    • Domaines : ❤️ Vie / 🪷 Nature
    • Sauvegarde : Sans
    • Description : Une vitalité persistante soutient le personnage. Les blessures récentes évoluent plus lentement, comme si son corps luttait activement pour empêcher leur aggravation.
    • Effet : A chaque rounds la gravité des blessures récentes (non stabilisée) augmente 2x moins

    Léger

    • Type : Positif
    • Domaines : 🌪️ Air / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : Le personnage se déplace avec une légèreté inhabituelle. Son équipement semble peser moins lourd, facilitant ses mouvements et réduisant la contrainte physique qu’il impose.
    • Effet : La charge de l'équipement du personnage est réduit de 1/4 (1/2)

    Précognition

    • Type : Positif
    • Domaines : 🧠 Esprit / 👁️ Vision
    • Sauvegarde : Sans
    • Description : Le personnage perçoit des fragments d’événements avant qu’ils ne surviennent. Cette anticipation surnaturelle élimine toute maladresse possible dans ses actions.
    • Effet : Seuil des maladresses +5 (impossible)

    Imperturbable

    • Type : Positif
    • Domaines : 💧 Eau / 🌿 Flore
    • Sauvegarde : Sans
    • Description : L’esprit du personnage reste parfaitement fluide. Aucune entrave extérieure ne parvient à ralentir son initiative ou à perturber son rythme d’action.
    • Effet : Si l'initiative doit être réduite par un effet alors la réduction est diminuée de 2 (pas de réduction quoi qu'il arrive)

    Hypertrophie

    • Type : Positif
    • Domaines : 🌀 Chaos / ⚕️ Corps
    • Sauvegarde : Sans
    • Description : Le membre affecté se développe de manière excessive et instable. Cette croissance chaotique permet au personnage d’en faire usage malgré ses limitations, en atténuant partiellement les désavantages liés à son état.
    • Effet : Le personnage est capable de faire usage du membre affecté en réduit les désavantage de 1 (2)

    Inépuisable

    • Type : Positif
    • Domaines : 🌿 Flore / ❤️ Vie
    • Sauvegarde : Sans
    • Description : Une endurance végétale soutient le personnage. Les effets de l’usure prolongée sont atténués, comme si son corps se régénérait lentement mais constamment.
    • Effet : Les dégats d'attritions subit par le personnage sont réduits de 1 pour 3 (2)
